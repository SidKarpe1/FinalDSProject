import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
df = pd.read_csv('ds_salaries.csv')
# function that calculates the quantity of values in a table
def frequency(list1):
  dict1 = {}
  for i in list1.to_numpy():
        if i not in dict1:
          dict1[i] = 1
        else:
          dict1[i] += 1
  return dict1

def bar_graph(data): # creating a function that makes a bar graph for any column of data
    dict1 = frequency(df[data])
    categories = dict1.keys() # categories as list
    values = dict1.values() # frequencies as lists
    plt.bar(categories, values, color='skyblue')
    plt.xticks(rotation=90)
    plt.xlabel(data)
    plt.ylabel('Frequency')
    plt.title('Bar Graph')
    plt.show()

#calling the bar graph functions
bar_graph('experience_level')
bar_graph('employment_type')
bar_graph('job_title')

Question 3: What is the best position in data science in terms of salary

To find the best job position, well look at the entry job positions to see which job you should try to start out as. In order to do this, we need to take a few steps. Our first step is to locate all beginner level employees. Our second step is to make a table with these employees job titles and salaries. Since this table has multiple people with the same job title, well group people will the same position into one column and average their salaries. Our last step is to reorder the table by descending salary to find the highest paid beginner positions.

In[5]:

Question 7: Has the average salary increased over time

For this question, were going to create a graph since it will be easier to vizualize how salaries increase. In order to do this, were first going to create two columns, one with the years as a string and the other with the mean salary of each year as an integer. Well then use a for loop to create an array with all the salaries for each year and store them in a dictionary as a value assigned to a year as a key. Finally, we can use matplotlib to create a line graph to show us how the salaries change over time.

In[34]:
import numpy as np
import numpy as np
import matplotlib.pyplot as plt
df = pd.read_csv('ds_salaries.csv')
#query: find unique years and turning them into strings
years = df['work_year'].unique()
years = [str(i) for i in years]
values = df['work_year']
#creating a dataframe with years as columns and salaries as entries
tempdict = {}
for i in years:
  tempvar = df.loc[df['work_year']==i]['salary_in_usd'].to_numpy()
  tempdict[i] = tempvar
# query: group by position with mean salary
sal = df.groupby('work_year')['salary_in_usd'].mean().to_numpy()
# creating a plot with years as x value and salaries as y values
plt.plot(years, sal)
plt.xticks(rotation=45)
plt.xlabel('years')
plt.ylabel('salary')

Now, we'll create a dataframe with a column that shows the rate of change over the years. To do this, we start by putting these columns in a dictionary and turning it into a dataframe. Then, we can use the pandas function .pct_change() to calculate the rate of change.
